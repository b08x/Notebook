## pandoc markdown to pdf 
#pandoc #latex0


[customizing pandoc](https://learnbyexample.github.io/customizing-pandoc/)


```bash
pandoc resume.md -f gfm \
	-V linkcolor:blue \
	-V geometry:a4paper \
	-V geometry:margin=1cm \
	-V margin-right=1cm \
	--pdf-engine=xelatex \
	-o resume.pdf \
	-V pagestyle=empty
```
